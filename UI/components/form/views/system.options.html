System Options

Select Domain

Indicate which CMS domain CMS you would like to use. This means you will be searching and manipulating CMS items in that domain.

Logout

This machine has your login data stored. It also has stored the items you've been looking at and other data that you have pulled from the database during this session. If you feel your session data is sensitive, clear both your login and session data. If you want to keep your session data so that you can resume from where you left off, just clear your login data.

Subitem UI Control

Within an item you may have several subitems. Those can be arranged in tabs, or within an accordion. Choose which here.

Notification Delay

Sets the number of seconds a system notification will be displayed prior to disappearing.

Search Results Per Page

The number of results that will appear on a single page after you perform a search.

Component Caching

As components are loaded their packaged files can be cached, such that reloading the component pulls the packaged files from cache instead of fetching them remotely via a service call. This speeds up responsiveness, and should always be on, except when debugging, where cache-busting can helpfully limit the amount of manual cache refreshes necessary.
